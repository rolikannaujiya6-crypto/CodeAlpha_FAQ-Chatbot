# 🤖 FAQ Chatbot

A simple FAQ Chatbot built using Python and Streamlit. This chatbot answers frequently asked questions by matching the user's input with predefined FAQs.

## 📌 Features

- Simple and user-friendly interface
- Answers frequently asked questions
- Uses Python's difflib for question matching
- Displays the best matching answer
- Easy to customize with new FAQs

## 🛠️ Technologies Used

- Python
- Streamlit
- difflib

## 📂 Project Structure

```
FAQ_Chatbot/
│── app.py
│── faqs.py
│── requirements.txt
│── README.md
```

## ▶️ Installation

1. Clone the repository:
```bash
git clone https://github.com/rolikannaujiya6-c/CodeAlpha_FAQ-Chatbot.git
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the project:
```bash
streamlit run app.py
```
